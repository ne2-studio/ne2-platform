# ADR-0008: ASP.NET as Default Backend Stack

Status: Accepted

## Context

The primary maintainer has extensive experience with ASP.NET.

## Decision

ASP.NET is the default backend technology for new applications.

See [Supported Application Types](../docs/04-application-architecture.md#42-supported-application-types).

## Consequences

* Faster development.
* Higher maintainability.
* Consistent backend architecture.
