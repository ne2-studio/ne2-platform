# ADR-0007: React as Default Frontend Stack

Status: Accepted

## Context

Most AI-assisted development tools generate React applications.

## Decision

React and Vite are the default frontend stack.

See [Supported Application Types](../docs/04-application-architecture.md#42-supported-application-types).

## Consequences

* Faster project bootstrapping.
* Better compatibility with AI-generated code.
* Frontend consistency across projects.
